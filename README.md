## getting started

### test

Run [jest] for all .spec.js/.spec.ts in `src/`.

```console
$ npm run test
```

### build

Compile with [tsc] for all js/ts in `src/` into `dist/`.

```console
$ npm run build
```

### watch(continuous build)

Watch with [tsc] for all js/ts in `src/` directory into `dist/`.

```console
$ npm start
```
